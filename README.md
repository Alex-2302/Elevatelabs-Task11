# Task 11: Phishing Attack Simulation & Detection

## Internship: Cyber Security Internship  
## Tool Used: GoPhish (Kali Linux)

---

## Objective
To understand phishing attacks by simulating a controlled phishing campaign using GoPhish and analyzing user behavior, detection methods, and prevention strategies.

---

##  Tools & Technologies
- Kali Linux
- GoPhish
- Mailtrap (SMTP Testing)
- VirtualBox

---

##  Task Workflow
1. Studied phishing attack concepts
2. Created a phishing email template
3. Designed a fake login landing page
4. Configured SMTP using Mailtrap
5. Launched a phishing campaign
6. Tracked email opens, clicks, and credential submissions
7. Identified phishing red flags
8. Documented prevention techniques

---

##  Results
- Phishing emails were successfully delivered
- User interactions were tracked
- Credentials submission was captured
- Campaign analytics were reviewed

---

## Final Outcome
Improved understanding of social engineering attacks and phishing detection/prevention techniques.
